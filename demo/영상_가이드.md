# 평생곁에 — 5분 데모 영상 제작 가이드

> 결론 먼저: **콘텐츠는 부족하지 않다. 오히려 7개 탭이라 5분을 넘길 수 있다.** 아래 타임라인대로 "보여줄 것만" 골라 찍으면 딱 5:00.

---

## 0. 녹화 준비 (한 번만)

- **브라우저: Chrome 권장** (음성 읽어주기 TTS·말로 질문 STT가 Chrome에서 동작).
- 브라우저 **F11 전체화면**, 확대 100~110%(Ctrl+0).
- **녹화 툴(무료, 택1)**
  - Windows 기본: **Win + Alt + R** (Xbox Game Bar) — 가장 간단.
  - 더 깔끔하게: **OBS Studio**(무료) — 1920×1080 / 30fps / 화면+마이크 동시.
- 마우스 강조: PowerToys 'Mouse Highlighter' 또는 Game Bar 옵션 켜면 클릭이 잘 보임.
- **알림 끄기**(방해금지), 바탕화면 정리, 데모 외 탭 닫기.
- **상태 초기화**: 녹화 직전 새로고침(Ctrl+F5) → 대시보드 카운터 0부터 시작.

> ⚠️ **촬영 순서 핵심**: 본부 상황판의 ₩금액·지도·로그는 *앞에서 거래를 막을수록* 쌓입니다.
> 그래서 **사기 방어 → 실시간 감시를 먼저** 하고 **본부 상황판을 뒤에** 보여줘야 숫자가 차 있습니다. (아래 타임라인이 그 순서)

## 1. 5분 타임라인 (내레이션 + 화면 동작)

### ① 0:00–0:30 · 오프닝 (탭: 홈)
- **화면**: 홈 진입. 통계 4종(30.6% / 5배 / 55.3% / −23.4%) 카운트업.
- **내레이션**:
  > "전북·전남은 전국에서 가장 빠르게 늙어갑니다. 점포는 23% 사라졌고, 어르신은 디지털에서 소외되고, 노후자금을 노린 사기는 10년 새 5배 늘었습니다. 그래서 우리는 자산관리를 다시 정의했습니다 — **불리는 게 아니라, 지키는 것**."

### ② 0:30–2:00 · 히어로: 송금 직전 30초 (탭: ① 사기 방어) ★가장 중요
- **화면 동작**:
  1. 시나리오 **박OO** 선택 → '송금하기' → **안전🟢 통과**. (평소 아들에게 보내던 패턴)
  2. **📞 전화 장면부터 재생** 클릭 → 검찰 사칭 자막 흐름 → '전화 끊고 송금' → **위험🔴 차단** 시트.
  3. 오른쪽 **AI 판단 로그(1층 룰+2층 베이스라인)** 와 **베이스라인 차트(이번 거래만 빨강)** 짚기.
  4. **직접 만들기** 토글: 같은 300만원에서 '처음 보는 사람'·'심야'로 바꾸면 점수가 치솟는 것 시연.
- **내레이션**:
  > "같은 300만원입니다. 평소 아들에게 보내던 박 어르신은 통과. 하지만 검찰 사칭에 속아 낯선 계좌로 보내는 김 어르신은 — 송금 직전, 은행이 **먼저** 멈춰 세웁니다. 금액이 아니라 **'이 어르신다움'**, 평소 거래 패턴 대비 이탈로 판단하기 때문입니다."

### ③ 2:00–2:40 · 쉬운말 자산 케어 + 접근성 (탭: ② 자산 케어)
- **화면 동작**: "주택연금 받으면 기초연금도?" 클릭 → 답변 + **🔊 읽어주기**(음성 재생). → "ELS 가입해도 돼?" → **가드레일**(단정 보류→사람 연결). → 상단 **A+ 글씨 크게 / 고대비** 토글.
- **내레이션**:
  > "어려운 금융은 쉬운 말로, 음성으로 들려줍니다. 그리고 모르면 추측하지 않고 상담사에게 연결합니다 — 금융의 생명인 환각 방지죠. 글씨 크기·고대비까지, 어르신을 위한 설계입니다."

### ④ 2:40–3:10 · 선제 알림 (탭: ③ 선제 알림)
- **화면 동작**: 알림 카드 4개(사기 차단/병원비/예금 만기/생신) 보여주고, 1~2개 **처리** 버튼 클릭 → 토스트.
- **내레이션**:
  > "에이전트는 물어봐야 답하는 챗봇이 아닙니다. 만기·의료비·이상거래를 스스로 지켜보다 **먼저** 말을 겁니다."

### ⑤ 3:10–3:55 · 실시간 감시 (탭: 🛰 실시간 감시)
- **화면 동작**: **'실시간 감시 시작'** 클릭 → 어르신들 거래가 흐르며 대부분 🟢통과, 위험 건은 **자동 🛑차단**. 10초쯤 흐르게 둠.
- **내레이션**:
  > "전북·전남 어르신들의 거래를 24시간 스스로 지켜봅니다. 대부분은 조용히 통과하고, 위험한 한 건은 버튼 없이 자동으로 차단됩니다. 이것이 에이전트의 정의입니다."

### ⑥ 3:55–4:40 · JB 본부 상황판 (탭: 🏦 JB 본부 상황판) ★심사 임팩트
- **화면 동작**: 앞에서 쌓인 **₩ 막은 송금액**, **차단 건수**, **전북·전남 지도**, **차단 로그**, **ROI** 짚기.
- **내레이션**:
  > "이건 고객이 아니라 **은행이 보는 화면**입니다. 보이스피싱 한 건 평균 피해는 530만원. 막을 때마다 고객의 노후자금과 함께 JB의 FDS 분담·민원·평판 비용을 절감합니다. 지방은행에게 이건 **비용이 아니라 투자**입니다."

### ⑦ 4:40–5:00 · 클로징 (탭: ④ 임팩트·기술)
- **화면 동작**: 아키텍처/기술 스택 1컷 → 확장 로드맵.
- **내레이션**:
  > "검증된 RAG 위에 개인화 FDS를 얹고, HyperCLOVA X로 온프레미스 배포까지. 평생곁에가 전북·전남 어르신의 노후자금을, **평생 곁에서 지키겠습니다**."

## 2. 촬영 팁

- **한 번에 다 찍지 말 것**: 탭별로 끊어 여러 번 찍고 합치면 실수 줄어듦(편집: 클로바 노트/Clipchamp/CapCut 무료).
- **마우스는 천천히**, 핵심 숫자(₩, 신호등, 점수)에 1~2초 멈춤.
- **음성 내레이션**은 또박또박. 라이브가 어려우면 화면만 찍고 나중에 보이스 더빙.
- **분량 초과 주의**: 7탭 다 보여주면 5분 넘음 → 위 타임라인의 동작만. 실시간 감시는 10초면 충분.
- 마지막에 **0.5배속 구간 없이** 자연스럽게, 5:00 안에 끝나도록 트리밍.
