## 0. 핵심 전략 요약 (TL;DR)

1. **판이 우리에게 유리하다.** 예선은 "돌아가는 데모"가 아니라 **기획 문서 2종**으로 겨룬다. 즉 빌드 완성도가 아니라 **아이디어의 날카로움 + 문제정의의 데이터 무장 + 기대효과 스토리텔링**이 승부처. 솔로에게 유리한 구조.
2. **차별화 각 = 역(逆)포지셔닝.** 대부분 팀은 WM을 "2030 자산증식 로보어드바이저"로 푼다. 우리는 정반대로 **JB의 실제 핵심 고객(전북·전남 거주 5060+ 고령층)** 을 정조준한다.
3. **JB 정체성과 정확히 일치.** JB는 대형 시중은행이 아니라 **지방은행(전북·광주은행) + JB우리캐피탈**. 영업권이 초고령 지역. → "이 팀은 JB를 진짜 이해했다"가 심사 임원에게 정서적으로 꽂힌다.
4. **사기방어는 '통화 탐지'가 아니라 '거래 차단'으로.** 실시간 통화 탐지는 이미 통신사(KT·LGU+·SKT) 영역. 은행의 고유 무기는 **계좌·거래 데이터(FDS)** 다. "이체 직전 개입"은 은행만 할 수 있고, 2024 FDS 가이드라인이 고령자 시나리오를 명시 → 정합성·차별성 동시 확보.
5. **기술 스택을 JB의 실제 방향과 정렬.** JB는 2025.12 **네이버클라우드(HyperCLOVA X·AICC)** 와 제휴. 제안서에 HyperCLOVA X 기반 RAG/온프레미스 sLLM을 그리면 "현실적이고 JB와 바로 붙는" 인상을 준다.

## 2. 시장·도메인 리서치 (= 슬라이드 2 "문제정의" 탄약고)

### 2.1 JB 영업권은 대한민국 초고령화의 최전선
- **전남 고령화율 30.6% — 전국 1위**, **전북 25.4%**(2025 고령자 통계). 65세+ 비중 20% 초과(초고령사회)는 전국 9곳: 전남·경북·강원·전북·부산·경남·충북·충남·대구.
- 통계청은 **2038·2052년에도 전남이 고령인구 비중 1위** 유지 전망 → 구조적·장기적 시장.
- (단, **광주는 17.9%로 낮음** → 제안서는 "전북·전남 중심" 포지셔닝이 정확. 광주는 도시형 디지털 격차 각도로 별도 서술)
- 출처: 통계청 2025 고령자 통계, KOSIS

### 2.2 점포는 사라지고, 고령층은 창구에 남았다
- 국내 은행 점포: 2012년 7,835개 정점 → **2025년 약 5,449~5,523개**(연평균 약 100개 폐쇄). 2024.9(5,620) → 2025.9(5,449), 1년새 171곳(3%) 감소.
- **전북 일반은행 점포: 2011년 149개 → 2023년 114개, 35개(23.4%) 감소**.
- 은행 점포 이동거리 20km 초과 상위 30개 지역 중 **26곳이 65세+ 비중 20% 초과 초고령 지역**(강원·전남·경북 최대 27km).
- 지방·농어촌은 고령화 속도가 빠른데 **여전히 창구 거래 의존 고령층 비중이 높다** → 점포 축소의 충격이 가장 큰 집단.
- 출처: 전국은행연합회, 금감원, 그린포스트코리아, 전북일보, 농민신문 등

### 2.3 디지털 정보격차 — 모바일뱅킹으로 못 흡수되는 사람들
- 2024 디지털정보격차 실태조사(NIA): 고령층 디지털정보화 수준 **71.4%**(4대 취약계층 중 최저).
- 세부: 접근 95.3% / **역량 55.3%**(절반 수준) / 활용 73.8% → **기기는 있으나 '쓸 줄'을 모른다**가 핵심 병목.
- 70대+ 스마트폰 보유율 전년比 +6.5%p → **디바이스는 깔렸다. 이제 '쉬운 인터페이스'가 관건** (우리 서비스의 존재 이유).
- 출처: 한국지능정보사회진흥원(NIA) 2024 디지털정보격차 실태조사

### 2.4 고령층을 정조준하는 금융사기 — 폭증 중
- 2025년 1분기 보이스피싱 5,878건, 피해액 **3,116억원**(전년 동기 대비 건수 +17%, 피해액 2.2배). **연간 피해 사상 첫 1조원 돌파 전망**.
- **60대+ 단일 연령대 피해 비중 30.6% — 전 연령 1위**. 60대 피해 2016→2025 **약 5배 급증**(1,261→5,801건). 50대+가 전체 피해의 **53%**(2023 32%→2024 47%→2025 53%, 매년 상승).
- 수법: **기관사칭형 51%**(검·경·금감원 사칭) — 고령층의 "권위 순응" 심리 악용. 사회적 고립·저학력·금융지식 부족 고령자일수록 취약.
- 출처: 경찰청·금감원, 세계일보, 한국시니어신문, 농민신문 등

### 2.5 규제도 "고령자 보호 + 거래단 탐지"를 요구한다
- **2024 FDS(이상금융거래탐지시스템) 운영 가이드라인**(금감원·금융보안원, 2024.1 시행): FDS 부실 은행에 피해액 일부 분담 권고.
- 가이드라인 제시 시나리오에 **"미성년자·고령자 보유 계좌에서 단시간 내 미거래 계좌로 소액 다수 이체 탐지"** 명시 → 우리 '거래단 사기방어'가 규제 흐름과 정확히 일치.
- 금융소비자보호법상 고령 금융소비자 보호(적합성·설명의무 강화)와도 연결.
- 출처: 금융보안원, KDI 경제교육·정보센터, Criminal IP 블로그

> **문제정의 한 문장(제안서용)**: "전북·전남은 전국에서 가장 빠르게 늙어가는데, 점포는 23%가 사라졌고, 고령층의 디지털 역량은 55%에 머물며, 이들을 노린 금융사기는 10년간 5배 늘었다. 대형은행이 떠난 이 자리에 남은 사람들을 지키는 것이 지방은행 JB의 존재 이유다."

## 3. 경쟁 환경 분석 (= 차별성 방어 논리)

### 3.1 금융권 시니어 서비스 — 있지만 '결'이 다르다
- **KB라이프 'KB골든라이프' 플랫폼(2026)**: 노후자금·건강·치매/요양돌봄·제휴혜택 4대 큐레이션. KB금융 **휴머노이드 로봇 '피지컬 AI' 돌봄**(국내 금융권 최초).
- **신한라이프**: 생성형 AI **상담요약**(상담사-고객 통화 실시간 요약), **'SOL메이트 시니어 콜센터'**, 자회사 신한라이프케어(프리미엄 요양원 'SOLACE').
- **차이점/빈자리**:
  - 이들은 대부분 **보험·요양(라이프케어 산업) 중심**이거나 **콜센터 보조** 성격. → "은행 거래·계좌 데이터를 활용한 일상 금융 + 사기방어 + 선제 개입"을 통합한 **에이전트**는 비어 있다.
  - **지방은행 영업권(초고령 지역) 맞춤**은 아무도 안 한다. 대형사는 전국·도시 중산층 타깃.
  - → **우리의 빈자리**: "지방 거주 고령 고객의 *은행 거래 일상* 그 자체를 지키는 선제적 에이전트".

### 3.2 보이스피싱 탐지 — '통화 탐지'는 이미 레드오션 (그래서 우회한다)
- **통신사 실시간 통화 기반 탐지**(2025.7 과기정통부 실증특례): **KT** 2025년 3천여 건 예방, 탐지 정확도 90.3%(1Q)→**97.2%(4Q)**. **LGU+ '익시오(ixi-O)'**(안티딥보이스 포함), **SKT 에이닷**, 삼성전자·이통3사 단말 연계.
- **민관 공동대응 앱**: **시티즌코난·피싱아이즈**(인피니그루) — 악성앱·원격제어앱·스미싱·URL 탐지.
- **전략적 함의**: 통화 음성 탐지는 통신사·전문앱이 선점 + 은행은 통화를 도청할 수도 없다. → **우리는 통화 탐지를 직접 하지 않는다.** 대신 **"이체·출금이 일어나는 순간"의 거래 맥락**(평소와 다른 수취인, 단시간 다수 이체, 고령 계좌 야간 대액 이체 등)에서 **은행만 할 수 있는 개입**을 한다. 필요 시 통신사 탐지 신호를 *입력으로 받는* 연계 구조로 설계(경쟁 아닌 보완).

### 3.3 경쟁 종합 — 우리의 3중 해자(moat)
1. **데이터 해자**: 은행 거래·계좌 데이터 → 통신사·헬스케어 앱이 못 가진 것.
2. **지역 해자**: 전북·전남 초고령 영업권 밀착 → 대형사가 안 하는 것.
3. **통합 해자**: 자산케어 + 사기방어 + 쉬운말 비서 + 선제 알림을 **하나의 에이전트**로 → 분절된 기존 서비스가 못한 것.

## 4. JB금융그룹 전략 정합성 (= 슬라이드 7 + 심사 공감 포인트)

- **네이버클라우드 제휴(2025.12)**: 계열 3사(광주은행·전북은행·JB우리캐피탈)가 네이버클라우드와 AI 금융 혁신 MOU. 협력 분야 = **HyperCLOVA X·AICC** 기반 과제 발굴, AI·빅데이터·클라우드 고도화, **금융 특화 모델 공동 개발**. → 우리 기술 스택을 HyperCLOVA X 기반으로 그리면 "JB와 즉시 정합".
- **기업여신 AI**: 상담→심사→사후관리 전반 AI 단계 검토. (참고: 우리 주제는 개인·리테일이지만, 동일한 "AI로 휴먼 업무 보강" 철학.)
- **공동전산센터(광주, 2025.10)**: 전북·광주은행 인프라 통합 → AI 디지털 금융 고도화 기반. **온프레미스/그룹 클라우드** 배포 시나리오의 현실성 근거.
- **그룹 AX/DX 본격화(2026.2)**: 'NewTech+비즈니스' 임직원 경진대회 등 전사 AI 적용 전략. 본 공모전도 그 연장선 → **JB가 진짜로 AI 인재·아이디어를 사업화하려는 의지**가 강함 = 실현 가능성·사업성 강조가 먹힌다.

### 5.2 보이스피싱/금융사기 탐지 — 논문·코드 근거
**한국어 특화 (직접 활용 가능):**
- **KoBERT 기반 통화내용 분석 보이스피싱 탐지** — 최대 정확도 **99.6%**(Boussougou & Park, 2022). 형태소 분석 토큰화 + 사전학습 KoBERT.
- **KoBERT + CNN-LSTM 듀얼 파이프라인** — 텍스트(문맥 위험도) + 음성(합성음성 판별, so-vits-svc 데이터). 실제 피싱 샘플 **94.9%** 탐지.
- **KoBERT 기반 보이스피싱 예방 서비스**(2023) — 위험도 평가 + 예방 교육 + 대처법 제공.

**공개 데이터셋·GitHub (제안서 부록·PoC용):**
- `selfcontrol7/Korean_Voice_Phishing_Detection` — **KorCCVi(Korean Call Content Vishing) 데이터셋** + KoBERT·연합학습 구현.
- `kimdesok/Text-classification-of-voice-phishing-transcripts` — 실제 통화 전사 2,927건 + 정상 통화 2,232건, KoBERT 전처리·학습 코드.
- GitHub Topic: `voice-phishing`.

**최신 글로벌 흐름 (기술 정당성):**
- *"Detecting Voice Phishing with Precision: Fine-Tuning Small Language Models"* (arXiv:2506.06180, 2025) — **소형 LM 파인튜닝**으로 정밀 탐지 → 온프레미스/저비용 배포 근거.
- *"A Multimodal Voice Phishing Detection System Integrating Text and Audio Analysis"* (MDPI Applied Sciences, 2025) — 텍스트(KoBERT+self-attention) + 오디오(MFCC+CNN-BiLSTM) 융합.
- ASR(STT) → 텍스트화 → 실시간 분류 파이프라인이 표준화됨.

> **우리 설계 포인트**: 통화 탐지는 **연계(외부 신호 수신)** 로만 두고, **자체 핵심은 '거래단 이상탐지 + 선제 개입'**. 보이스피싱 모델은 "우리가 직접 다 만든다"가 아니라 "검증된 오픈모델/통신사 신호를 오케스트레이션한다"로 가면 솔로 현실성↑, 차별성↑.

### 5.3 이상거래 탐지(FDS) — 룰 + ML
- **2024 FDS 가이드라인**의 고령자 시나리오(소액 다수 이체 등)를 **룰 베이스**로 즉시 구현 가능(설명가능·현실적).
- 고도화: **GNN(그래프 신경망)** 기반 카드/거래 사기탐지가 최신 SOTA — 단일 거래가 아닌 **계좌 간 관계·사기링** 포착(detectGNN, CaT-GNN 등 2024~2025). 클래스 불균형 대응(SMOTE 등), 연합학습(프라이버시 보존)도 활발.
- **제안서 전략**: MVP는 **설명가능한 룰 + 가벼운 ML 이상탐지**(Isolation Forest 류)로, 향후 확장에 **GNN·연합학습** 명시 → "현실적 MVP + 야심찬 로드맵" 균형.

### 5.4 RAG / Agentic RAG / 가드레일 (할루시네이션 = 금융의 핵심 리스크)
- **Agentic RAG**: 정적 파이프라인을 넘어 *계획→검색→추론→비평(critic)→재작성→반영* 루프. **LangGraph**(상태기반 순환 그래프, 조건 분기, 체크포인트, human-in-the-loop)로 구현하는 게 2025~2026 표준.
- **할루시네이션 가드레일**: critic 에이전트 + reflection loop + introspection 프롬프트("데이터 없음 → 단정 대신 도구 호출/모름 응답"). 은행 챗봇이 RAG+휴먼감독으로 정확도 **25%→89%** 개선 사례.
- **금융 특화 보안**: "Semantic Firewalls for Secure Agentic RAG in Financial Chatbots"(2026) 등 금융 챗봇 안전장치 연구 진행.
- 참고 구현: `junfanz1/Cognito-LangGraph-RAG-Chatbot`(문서 검색 + 관련성 평가 + 웹검색 폴백).
- **우리 적용**: "답변마다 **법령·상품약관 조항/문서 근거를 인용**, 근거 없으면 **'모른다 + 상담사 연결'**" → 고령 고객 신뢰의 결정타.

### 5.5 한국어 금융 LLM — 벤치마크·모델 생태계 (기술 신뢰성 근거)
- **KRX-Bench**(FinNLP@LREC-COLING 2024): GPT-4 자동 생성 한국 금융 벤치마크(1,002문항).
- **₩on / KRX-Data**(arXiv:2503.17963): 한국 금융 NLP 베스트프랙티스, 오픈 리더보드(2개월간 1,119개 모델 제출, 상금풀 ~$42K) → 한국어 금융 LLM 경쟁이 매우 활발.
- **KFinEval-Pilot**(arXiv:2504.13216): 지식·추론·**독성/안전(규제준수·사기리스크·소비자보호)** 평가 — 우리 도메인과 정확히 겹침.
- **KorFinMTEB**: 한국 금융 임베딩 벤치(도메인 임베딩의 문화·의미 뉘앙스 가치).
- **시사점**: 범용 LLM보다 **한국 금융 도메인 임베딩 + 안전 평가**가 중요 → 우리 RAG 임베딩 선택·평가에 KorFinMTEB/KFinEval 인용 가능.

## 6. 제안 솔루션 — 4대 핵심 기능 (= 슬라이드 4 / 기능명세서 3)

| # | 기능명 | 설명 | 입력 → 출력 | 핵심 기술 | MVP 구현 |
|---|--------|------|-------------|-----------|----------|
| F1 | **생애주기 자산·생활 케어** | 연금/예금만기/의료비/상속·증여를 고령 눈높이로 안내·요약 | 거래내역·프로필·질문 → 쉬운말 안내 카드 | RAG(상품·제도 문서)+LLM | O(핵심) |
| F2 | **거래단 사기 방어(선제)** | 평소와 다른 이체·수취인·소액 다수 이체 등 탐지 → 송금 전 경고·차단 | 거래 이벤트 스트림 → 위험 신호등+개입 | FDS 룰+이상탐지 ML, (외부)통신사 신호 연계 | O(룰 PoC) |
| F3 | **쉬운말 금융비서** | "이번 달 왜 돈이 부족해?" 같은 자연어 질문에 거래 해설·용어 풀이 | 자연어 질문 → 음성+카드 응답 | STT/TTS+RAG+LLM | △(부분) |
| F4 | **선제 알림 & 사람 연결** | 만기·의료비·이상거래를 *먼저* 알리고, 필요 시 상담사/가족 핸드오프 | 이벤트 → 알림+원터치 액션 | 이벤트 트리거+에이전트 플래너 | X(설계) |

> "Agent"의 정체성 = **F2/F4의 선제성**(묻기 전에 먼저 행동) + **도구 사용**(RAG·FDS·알림·핸드오프). 단순 Q&A 챗봇과 결정적으로 다른 지점.
