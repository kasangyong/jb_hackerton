# 「평생곁에」 디자인 가이드
> 레퍼런스: **DRIFT 최종제출본** 의 디자인 언어를 표준으로 채택.
> 한 줄 컨셉: **"DRIFT의 다크 프리미엄 관제센터 톤 + 평생곁에의 따뜻함 1스푼."**
> 이 문서대로 7장 MVP 제안서를 리디자인한다.

---

## 0. 큰 방향 (3원칙)

1. **다크 모드 프리미엄.** 흰 배경 카드 도배 → **딥 네이비-블랙 배경** + 발광하는 액센트. 진지하고 고급스러운 인상 = 심사 가산점.
2. **모든 슬라이드 동일 해부구조.** 헤더·페이지번호·아이브로우·헤드라인·리드·콘텐츠·하단 takeaway·출처[S#]가 매 장 같은 자리. (DRIFT의 최대 강점)
3. **따뜻함은 죽이지 않는다.** 라이프케어 주제이므로 **어르신 사진 1컷 + 골드 액센트**로 인간미를 남긴다. DRIFT는 차가워도 되지만(해양 긴급), 우리는 사람을 지키는 서비스.

---

## 1. 컬러 토큰 (다크)

| 역할 | 토큰 | HEX | 용도 |
|---|---|---|---|
| 배경 (페이지) | `--bg0` | `#0A1422` | 슬라이드 바탕 (그라데이션 `#0A1422→#0E1F38`) |
| 카드 | `--bg1` | `#101E33` | 콘텐츠 카드 |
| 카드(raised) | `--bg2` | `#16263E` | 강조 카드·다이어그램 |
| 외곽선 | `--line` | `rgba(255,255,255,.08)` | 카드 보더 (얇게) |
| 본문 강 | `--ink` | `#EAF1FA` | 제목·핵심 텍스트 |
| 본문 약 | `--ink-soft` | `#8FA3BE` | 설명·캡션 |
| **HERO 액센트(브랜드/따뜻함)** | `--gold` | `#F6A23B` | 로고·핵심 키워드·"지킴" 강조·CTA |
| **DATA 액센트(긍정/기술)** | `--sky` | `#46C7E0` | 헤드라인 키워드·수치 강조·다이어그램 라인 |
| 안전 🟢 | `--safe` | `#2FD27A` | 통과·정상 |
| 주의 🟡 | `--warn` | `#F4B740` | 경고·시간 낭비 |
| 위험 🔴 | `--danger` | `#FF5A4D` | 차단·사망·치명 수치 |

**색 사용 규칙**
- 한 슬라이드에 **액센트는 2색까지**(보통 sky + gold). 나머지는 무채색.
- **gold = 사람·지킴·브랜드**, **sky = 데이터·기술·속도**, **red = 위협 수치**. 의미를 섞지 말 것.
- 신호등(🟢🟡🔴)은 다크 배경에서 **발광(glow)** 처리하면 고급스럽다.

## 2. 타이포그래피

| 용도 | 사양 |
|---|---|
| 헤드라인 | Pretendard **ExtraBold~Black**, 큰 크기. **핵심 단어만 `--sky`(또는 위협수치 `--danger`)** 컬러. |
| 아이브로우/섹션태그 | **레터스페이싱 넓은 대문자**, 모노 느낌 (예: `02 PROBLEM · 문제 정의`). `--sky` 컬러, 작게. ← DRIFT 시그니처 |
| 라벨·코드·STEP·[S#] | 모노/대문자 레터스페이싱 (`STEP 01`, `L1 PHYSICS`, `[S1]`). `--ink-soft`. |
| 본문 | Pretendard Regular/Medium, `--ink-soft`, 줄간 1.5. |
| 거대 스탯 숫자 | Black, 초대형. **단위는 작게 + 액센트색**(예: 30.6**%**, 530**만원**). |

> **금지**: Arial, 단어 중간 줄바꿈(orphan), 플레이스홀더 텍스트 노출.

## 3. 슬라이드 해부구조 (매 장 동일)

```
┌───────────────────────────────────────────────────────────┐
│ 평생곁에 · [섹션명]                              02 / 07   │  ← 러닝 헤더 + 페이지번호
│ ──────────────────────────────────────────────────────────│  ← 얇은 풀폭 라인
│                                                            │
│ 02 PROBLEM DEFINITION · 문제 정의           ← 아이브로우(sky, 레터스페이싱)
│ 지킬 자산을 잃으면, 관리할 자산도 없다       ← 헤드라인(Black, 키워드 sky/red)
│ 남들은 자산을 '불리는' WM을…                  ← 리드 1줄(ink-soft)
│                                                            │
│ [ 콘텐츠 영역 — 카드/다이어그램 ]                          │
│                                                            │
│ 🛡️ 한 줄 takeaway 메시지                  [S1][S2]  ← 하단 결론 + 출처
└───────────────────────────────────────────────────────────┘
```
- **아이브로우 번호**(02)와 페이지번호(02/07)를 일치시켜 내비게이션 감각.
- **하단 takeaway**는 매 장 1줄, 앞에 🛡️(평생곁에의 아이콘, DRIFT의 ⚓ 대응).

## 4. 시그니처 컴포넌트 (DRIFT에서 그대로 차용)

1. **거대 숫자 스탯카드** — 상단 컬러 라벨칩 + 초대형 숫자(단위 액센트) + 캡션 + `[S#]`.
   - 예: `30.6%` 전남 고령화율 · `530만원` 평균 피해 · `5배↑` 60대 피해.
2. **좌측 비주얼 + 우측 3카드** — 다이어그램/그래프 왼쪽, +설명 카드 3개 오른쪽.
   - 우리 적용: 슬라이드3 아키텍처, 슬라이드4 베이스라인 차트.
3. **INPUT → ENGINE → OUTPUT 3열** (사이 화살표) — 슬라이드3/5.
4. **번호 스텝 타임라인** (STEP 01–04, 연결선+점) — 사용자 시나리오/플로우.
5. **BEFORE / AFTER 비교 표** — 우리(평생곁에) 열을 sky로 하이라이트. ← 슬라이드7 ROI에 사용(§6).
6. **제품 스크린샷 in 디바이스 프레임 + 번호 콜아웃(①②③④)** — **데모 캡처를 그대로** 폰/브라우저 목업에 넣고 1·2·3·4 포인트 지시. (DRIFT의 제품화면 슬라이드 방식)
7. **출처 슬라이드 [S1]–[S#]** + 본문 곳곳 `[S#]` 각주 → "근거 있는 팀" 인상.
8. **한계 슬라이드(Honest by Design)** — 한계 3~4개 + 대응. 여유 있으면 추가(신뢰도 급상승).

---

## 5. 따뜻함 주입 규칙 (DRIFT와 다르게 할 것)

- **표지**: 다크 배경 위에 **전북·전남 어르신/동네 실제 사진**(저작권 free)을 **그라데이션 오버레이**로 얹어 차갑지 않게. 우측에 폰 경고 미니카드.
- **gold 액센트**를 "지킴·사람" 맥락에 사용(로고 방패, "먼저 개입", "평생 곁에").
- 카피 톤: DRIFT는 군더더기 없는 작전체. 우리는 **작전체 + 한 끗의 따뜻함**("어르신", "평생 모은 노후자금").

## 6. 슬라이드 7 (기대효과) — ₩ ROI 리디자인 [필수 반영]

> 현재 슬라이드7은 "↑↓ 정성 표현"만 → **DRIFT의 BEFORE/AFTER 표 + KEY INSIGHT** 형식으로 교체. 심사위원(은행)이 보고 싶은 **₩ 숫자**를 넣는다.

**헤드라인**: `막은 만큼, 지킨 돈. 방치는 곧 비용이다.`

**BEFORE / AFTER 표** (우측 '평생곁에' 열을 sky 하이라이트)

| 지표 | 현재 · 방치 | 평생곁에 적용 | 근거 [S#] |
|---|---|---|---|
| 보이스피싱 1건 손실 | **평균 530만원** + 민원·평판 비용 | 송금 직전 **차단** | 경찰청·금감원 2025 1Q [S1] |
| FDS 분담 리스크 | 2024 가이드라인상 **은행 분담** | 예방 → 분담 **회피** | 금감원·금융보안원 2024 [S2] |
| 고령 고객 접근성 | 점포 **−23.4%**, 창구 의존 | **24시간** 비대면 케어 | 은행연합회 [S3] |
| 고령 채널 이탈 | 디지털 소외 → 이탈 | 쉬운말·음성으로 **유지** | NIA 2024 [S4] |

**거대 숫자 3종 (스탯카드)**
- `530만원` — 보이스피싱 1건 평균 피해 (= 막을 때마다 지키는 돈)
- `12,840명` — JB 보호 대상 고령 고객(전북·전남, 가정)
- `24시간` — 점포 없어도 쉬지 않는 선제 감시

**KEY INSIGHT 바 (하단, sky)**:
> 보호 고객이 사기에 노출될 때마다 평생곁에가 1건을 막으면 **₩530만원의 고객 노후자금 + JB의 FDS 분담·민원·평판 비용**을 동시에 절감한다. 지방은행에게 이건 **비용이 아니라 투자**다. (포용금융·ESG 정합)

> ※ 데모의 **'JB 본부 상황판'**(막은 피해액 ₩ 실시간 카운터·전북전남 지도·차단 로그)을 이 슬라이드의 스크린샷으로 넣으면 효과 극대화.
