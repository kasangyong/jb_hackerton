> 이 문서는 **예선 제출 기능명세서(F1~F6) 대비 본선 시점 실제 구현에서 달라진 점을 누락·허위 없이** 기재한다.

| 기능 | 예선 명세 | 본선 현재 | 변경 유형 |

| **F3 쉬운말 비서** | △ 부분(STT/TTS 데모) | 변경 없음(여전히 부분) | 변경 없음 |
| **F4 선제 알림** | △ 목업 | **다채널 선제 개입 시각화(실감 시나리오) 추가** · 실연동은 여전히 설계 | 기능 보강 |
| **F5 가드레일** | O | 구현 유지 + **정확도 평가로 작동 검증** | 검증 강화 |
| **F6 본부 상황판** | 집계·시각화 | **관제 AI 브리핑(LLM) 추가** + **지도 SVG→카카오맵** | 기능 추가 |

## 1. 기능별 변경 상세

| 변경 일자 | 대상 기능 | 변경 전 (예선) | 변경 후 (본선 현재) | 변경 사유 |
|---|---|---|---|---|
| 2026.06.30 | **F2 거래단 사기 방어** | 2층 구조: (1층) FDS 가이드라인 룰 + (2층) 개인화 베이스라인(z-score) | **3층 구조로 확장** — (3층) **수취계좌 평판 조회** 추가. 사기 신고 이력 계좌면 점수와 무관하게 **즉시 완전 차단** (`blacklisted`). 소액·아는 패턴이어도 차단. | 대포통장(수취계좌) 신호를 더해 탐지 강화 |
| 2026.06.30 | **F6 JB 본부 상황판** | 막은 송금액·전북전남 보호 현황·차단 로그·ROI **집계·시각화** | **관제 AI 브리핑 추가** — EXAONE이 차단 로그를 본부 담당자용 보고체로 **자동 요약 + 권고 조치**(`/brief`). 로그 근거만 사용(환각 방지). | 은행 측 LLM 활용(직원 단순·반복 보고 자동화) |

| 2026.07.01 | **F1 생애주기 자산 케어** | 공개 금융문서 **6종** RAG(연금·예금·보이스피싱·의료비 등) | 문서 **7종으로 확대** — **전북은행 실제 공개 상품(JB 골든에이지·JB 리치 100, jbbank.co.kr) 추가**. 데모 예시 질문을 RAG가 실제 답하는 항목으로 교체(계좌 개인데이터 질문 제거). | JB 실상품 정합성, 데모–백엔드 일치 |
| 2026.06.30 | **F4 선제 알림 & 사람 연결** | 데모 목업(알림 카드) | **다채널 선제 개입 시각화 추가** — ‘실감 시나리오’로 앱 푸시·**ARS 자동전화(TTS)**·가족 알림 3중 채널을 분 단위로 재현. 실제 채널 연동은 여전히 설계 단계. | 무설치 앰비언트 개입 흐름 구체화 |
| 2026.06.30 | **F5 가드레일** | 규칙 기반 후처리(근거 없으면 사람 연결) | 구현 유지 + **간이 정확도 평가(eval)로 작동 검증** — 범위 밖 질문 가드레일 거부율 등 측정. | 신뢰성 정량 검증 |

## 3. 표현·포지셔닝 정정 (허위 방지 차원 — 기능 자체 변경은 아니나 명확화)

| 항목 | 예선 표현 | 정정된 표현 | 사유 |
|---|---|---|---|
| **모델 라이선스** | EXAONE-3.5 / HyperCLOVA X SEED 병기 | **PoC = EXAONE(연구·비상업) / 실배포 = HyperCLOVA X(상업 이용 가능, JB–네이버클라우드 제휴)** | EXAONE 상용 불가 → 실배포 모델 명확화 |
| **마이데이터(타행)** | ‘타행 계좌까지 보호 범위 확장’ | **실시간 차단은 자행(이체중계단)에서만**, 타행은 **통합 조회·사후 모니터링**(실시간 차단 아님) | 마이데이터는 주기적 호출 → 실시간 불가 |
| **학습 여부** | (명시 없음) | **모델을 학습(파인튜닝)하지 않음** — 검색·규칙·통계 기반. 학습형 고도화(파인튜닝·GNN)는 내부 실거래 필요 → 확장 | 데이터 규모 오해 방지 |

## 4. 변경 없음(정직 고지)

- **F3 쉬운말 금융비서**: 여전히 부분 구현(브라우저 Web Speech STT/TTS 수준). 본선까지 완성 예정 항목.
- **F4 실제 채널 연동**(AICC ARS·알림톡·FCM): 설계 단계(연동 대상은 은행 인프라). 데모는 시각화까지.
- **확장 항목**(Qdrant·Cross-Encoder Reranker·kiwipiepy·GNN·통신사 신호·마이데이터 실연동): 미구현, 로드맵 유지.

> 위 항목들은 "구현했다"고 **주장하지 않는다.** 구현 여부 표기(O/△/설계)는 실제와 일치한다.
